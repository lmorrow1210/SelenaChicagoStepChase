Pill-shaped action button; use for any tap target — primary actions use blue fill, rivalry actions use the red danger variant.

```jsx
<Button variant="primary" icon="sync">Sync steps</Button>
<Button variant="secondary">Maybe later</Button>
<Button variant="danger" icon="nemesis">Rematch</Button>
<Button variant="ghost" size="sm">Skip</Button>
<Button loading>Saving…</Button>
```

Variants: primary (blue fill / navy text), secondary (blue outline), danger (red fill / cream text), ghost (cream text, subtle hover fill). Sizes sm/md/lg. Supports `icon`, `iconRight`, `loading`, `disabled`, `fullWidth`.
